Ligne de commande

Ganttproject vous permet d'indiquer des instructions grâce à la ligne de commande.
Essayez "java -jar ganttproject.jar --help" pour afficher les paramètres utilisables.

Vous pouvez donner directement le nom du fichier XML à ouvrir.
Ex. : java -jar ganttproject.jar monprojet.xml

Une autre possibilité est d'exporter le projet en pages HTML, document PDF ou en images PNG/JPG.
Ex. : java -jar ganttproject.jar -html monprojet.xml directory pour exporter le projet en pages web dans la directorie spécifiée ;
Ex. : java -jar ganttproject.jar -pdf monprojet.xml monprojet.pdf pour exporter le projet en monprojet.pdf ;
Ex. : java -jar ganttproject.jar -png myproject.xml myproject.png pour exporter le projet en monprojet.png ;
Ex. : java -jar ganttproject.jar -jpg monprojet.xml monprojet.jpg pour exporter le projet en monprojet.jpg.

Avec l'exportation HTML vous pouvez spécifier une directorie XSL pour l'utilisation des arguments XSL.

Dans tous les cas le fichier projet peut être stocké sur le disque dur ou sur une adresse URL. Si l'adresse est protégée par un mot de passe, vous pouvez utiliser cette méthode :
Ex. : java -jar ganttproject.jar http://nomutilisateur:password@exemple.com/nomfichier
