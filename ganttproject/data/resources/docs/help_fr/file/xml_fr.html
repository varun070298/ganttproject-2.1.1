Format XML

Ganttproject utilise le format de fichier XML. Il est donc très simple de l'éditer avec un éditeur de texte standard.

Ce fichier XML contient les informations suivantes :

  • informations du projet
    • nom du projet ;
    • organisation ;
    • lien internet ;
    • description.
  • informations calendaires
    • jours du week-end ;
    • jours fériés.
  • informations sur les colonnes du tableau des tâches
    • nom et type ;
    • colonnes affichées et ordre d'affichage ;
    • colonnes personnalisées.
  • roles définis par l'utilisateur ;
  • tâches du projet ;
  • ressources du projet ;
  • affectation des ressources aux tâches ;
  • jours de congés des ressources ;
  • informations sur le zoom et la vue.

Les extensions des fichiers peuvent-être .XML ou .GAN. Les informations sont strictement les mêmes dans ces deux formats.
