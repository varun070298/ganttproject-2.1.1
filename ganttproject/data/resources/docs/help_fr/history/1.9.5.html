Release 1.9.5

  • il n'est plus possible de sauver / ouvrir dans le format GAN (utiliser une ancienne version pour sauvegarder en XML) ;
  • affectation des ressources aux tâches ;
  • toutes les classes tu JAR sont dans la package net.sourceforge.ganttproject ;
  • copier / coller de tâches dans l'arbre (problèmes si deux tâches ont le même nom, à éviter) ;
  • vérification d'un changement dans le projet avant de demander confirmation pour sauver ;
  • sauvegarde de la taille et de la position de la fenêtre dans le fichier d'options. Lors de l'ouverture, c'est paramètres sont concervés ;
  • ajout du mot modifié à la fin du titre de la fenêtre pour spécifier que le projet a été modifié ;
  • bogue corrigé : lors de la réouverture d'un fichier XML, la liste des ressources était ajoutée à la courante, maintenant elle est remplacée ;
  • bogue corrigé : glisser une tâche sur le calendrier (de droite à gauche) ;
  • bogue corrigé : correction de l'export XML des caractères <&, < et > (merci à Andreas Drechsler de l'avoir soumis) ;
  • possibilité de changer l'apparence de l'application (sauvegarder dans le fichier d'options).
