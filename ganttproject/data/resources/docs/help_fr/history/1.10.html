Release 1.10

  • export en CSV et début en xfig ;
  • modification de l'interface, correctifs sur les entrées sorties webdav. Selection multiples des tâches dans l'arbre pour des actions simultanées. Restauration du format de l'arbre (noeuds ouverts ou fermés) ;
  • modifications de l'interface graphique ;
  • possibilité d'export en CSV compatible avec les tableurs ;
  • correctifs avec les entrées sorties en WebDav ;
  • la sélection multiple dans l'arbre est désormais activée pour quelques fonctionalités ;
  • gestion de l'orientation des composants pour les languages RTL (Right To Left) ;
  • correction du bug #889441 ;
  • ajout d'un lien web pour le projet ou l'entreprise ;
  • importation d'une liste de tâche à partir d'un fichier texte. La fusion des tâches chargées avec le projet actuel est possible ;
  • envoi de courriel aux ressources ;
  • possibilité de définir un lien web pour chaque tâche ;
  • changement d'icônes (celle d'eclipse 3.0) ;
  • ajout de l'apparence plactic ;
  • possibilité d'avoir une barre d'outils avec des grandes icônes ou petites ;
  • possibilité d'avoir les boutons de la barre d'outils avec icônes seules, textes seules ou icônes et textes ;
  • les boîtes de dialogue des péférences et à propos sont standardisées ;
  • possibilité d'exporter directement en XFIG (basé sur un programme de Havel ;
  • possibilité de lier directement plusieurs tâches ;
  • multi-sélection gérer dans l'arbre des tâches pour plusieurs actions :
    • indenter et dédenter des tâches ;
    • monter et descendre des tâches ;
    • supprimer des tâches ;
    • lier et délier des tâches.
  • changement du logo GanttProject et de l'icône ;
  • sauvegarde de l'information éclaté/masqué des tâches ;
  • l'exportation au format CSV est compatible avec la plupart des tableurs comme OpenOffice Calc par exemple ;
  • WebDAV : correction de problèmes de compatibilité avec Microsoft® IIS et Xythos WFS ;
  • ajout d'un test d'une bar de statut threadée avec beaucoup d'informations.
