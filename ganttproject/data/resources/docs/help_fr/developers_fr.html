Pages pour développeurs

GanttProject est développé en language objet Java sous la plate forme Eclipse.
Pour tout connaitre sur la façon de récupérer les sources depuis le CVS, compiler le programme et l'éxecuter depuis Eclipse, vous pouvez consulter le manuel pour les développeurs disponible sur la page des developpeurs.

Un Wiki est également disponible avec différents articles sur le développement au sein de GanttProject.
