Utiliser GanttProject

Ganttproject est simple à utiliser à condition qu'une "Machine Virtuelle JAVA" soit installée sur votre poste de travail.

Sous un OS tel que Windows, lancez l'application par un double-clique sur le lanceur ganttproject.bat.

Sous un OS tel que Unix-Linux, lancez l'application en utilisant le fichier ganttproject.sh de cette façon :
sh ganttproject.sh
