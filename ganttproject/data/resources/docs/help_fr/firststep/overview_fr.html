Vue d'ensemble

Ganttproject est une application 100 % Java qui vous permet de planifier un projet selon la méthode du diagramme de Gantt. Ganttproject vous permet de découper facilement un projet en tâches, de visualiser les dépendances et de gérer les ressources.

Ecrit en Java, les fichiers JAR fonctionnent sous plusieurs systèmes comme GNU/Linux, Microsoft® Windows, MacOsX ... De plus, des installeurs pour différentes plate-formes sont prévus voire déjà disponibles !
