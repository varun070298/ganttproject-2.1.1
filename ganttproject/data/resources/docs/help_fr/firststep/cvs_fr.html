Où trouver le CVS ?

Les développeurs de GanttProject vivent dans le monde entier, aussi ils utilisent un serveur de code source CVS. Il est possible de télécharger la version actuelle en utilisant un "login anonyme" :
Essayez ces lignes de commande :
  $ export CVSROOT=":pserver:anonymous@cvs.sf.net:/cvsroot/ganttproject"
  $ cvs login

  (No password, just press enter.)

  $ cvs -z3 co ganttproject
Des informations complémentaires peuvent être trouvées dans le guide des développeurs, dont la dernière version est disponible à l'adresse suivante : http://sourceforge.net/project/showfiles.php?group_id=72728
