GanttProject - Стартиране на GanttProject

Ganttproject е лесен за употреба, ако имате инсталирана java virtual machine.

Под операциони системи от семейството на windows, приложението се стартира с двойно щракване на ganttproject.bat.

Под операциони системи от семейството на Unix-Linux, приложението се стартира, като се използва файла ganttproject.sh. Например:
sh ganttproject.sh
