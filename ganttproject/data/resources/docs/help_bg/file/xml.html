GanttProject - XML формат

Ganttproject използва XML файл формат. Той е лесен за редактиране с помощта на обикновен текстови редактор, ако желаете. В XML файлът са записани списъкът със задачите, списъкът със ресурсите, зависимостите, и връзките между задачи и ресурси.

Файлът може да бъде записан с разширения .xml или .gan
