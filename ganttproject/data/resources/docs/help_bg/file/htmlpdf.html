GanttProject - HTML/PDF

Ganttproject може също така да експортира проекта като HTML интернет страници. Различните интернет страници съдържат цялата информация относно проекта:
  • Проектна информация
  • Графики (Гант и ресурсни)
  • Списък на задачите с с информация за тях, като начална и крайна дати, продължителност, бележки...
  • Списък на ресурсите
За експорт в HTML GanttProject използва XSL шаблони за HTML преобразуване.
GanttProject използва по подразбиране XSL шаблона от jar файла.
Вие обаче можете да зададете друг xsl шаблон, като зададете папката, която го съдържа в диалога за настройките.
Фактически вие може да създавате собствени xsl шаблони за експорт в подходящо за вас форматирани html интернет страници.

За експортиране в PDF софтуерът използва FOP библиотека за експорт в pdf формат. Използва се и xsl технология за преобразуване.
Така можете да използвате PDF теми, като темата по подразбиране е включена в GanttProject jar файла.
