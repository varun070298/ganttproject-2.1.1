GanttProject - Команди от конзола

Ganttproject ви позволява да зададете някои параметри, ползвайки командна конзола.
Въведете java -jar ganttproject.jar --help за да видите възможните параметри.

Първо, можете да зададете директно xml файл за отваряне от GanttProject.
Например : java -jar ganttproject.jar myproject.xml

Втората възможност е да експортирате проект като HTML страници, PDF документ или PNG/JPG изображение.
Например : java -jar ganttproject.jar -html myproject.xml directory да експортирате проект като интернет страница в указана от вас папка.
Например : java -jar ganttproject.jar -pdf myproject.xml myproject.pdf да експортирате гант графика като myproject.pdf.
Например : java -jar ganttproject.jar -png myproject.xml myproject.png да експортирате гант графика като myproject.png.
Например : java -jar ganttproject.jar -jpg myproject.xml myproject.jpg да експортирате гант графика като myproject.jpg.

Когато експортирате проект като интернет страница, можете да укажете xsl папка ползвайки -xsl-dir argument.

Във всички тези случаи проекта може да бъде зададен като файл на локалния диск или URL. Ако URL-ът е защитен с парола, можете да се идентифицирате ето така:
Например : java -jar ganttproject.jar http://username:password@example.com/filename
